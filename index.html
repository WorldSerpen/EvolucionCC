Evolución:
Control de Calidad


256 a.C

El emperador Qin Shi Huang estandarizó la fabricación de armas, como ballestas y flechas, utilizando moldes y medidas precisas para garantizar uniformidad.

1456

Johannes Gutenberg introdujo inspecciones sistemáticas durante la impresión de la Biblia para garantizar la calidad de las páginas y la tinta.

1798

Eli Whitney implementó el sistema de partes intercambiables en la fabricación de mosquetes para el gobierno de Estados Unidos.

1875

Se establecieron los primeros laboratorios para probar la calidad de materiales como el acero y el hierro.

1924

Walter A. Shewhart desarrolló los gráficos de control para monitorear procesos y reducir variabilidad.

1950

Williams E. Deming impartió conferencias sobre control estadístico de procesos y mejora continua para el control de calidad.

1951

Joseph Juran publicó libro "Quality Control Handbook", que se convirtió en una referencia fundamental para la gestión de la calidad.

1980

El "Total Quality Management" se adoptó globalmente, integrando la calidad en todos los niveles de la organización.

1886

Motorola desarrolló la metodología Six Sigma para reducir defectos mediante el análisis estadístico y la mejora de procesos.

1987

Se publicó la norma ISO 9001, estableciendo estándares internacionales para sistemas de gestión de calidad.

2012

Empresas como Tesla comenzaron a utilizar sistemas de visión artificial para inspeccionar piezas.

2015

Se comenzó a utilizar Big Data para analizar patrones y predecir fallos en los procesos de producción
